# query-navigator

![query-navigator-cover](https://github.com/user-attachments/assets/20e90441-6460-4da7-90b9-956d5c85e305)

LLM chat queries navigator extension

## Supported Chats
- Gemini
- ChatGPT
- Perplexity
- Deepseek
- Kimi


## Load Extension

- clone repo

- Open `chrome://extensions` (Enable *Developer Mode*)

- Click in `Load Unpacked`

- Open `query-navigator`(this repo) and select `quert-extension` folder

- You can see **Q** in the extension, click on it and start using it.



## Features
- Refresh : Smooth refresh when moved to different chat or tab.
- Settings : You can Add or Updated existing websites.
- Select Query : You can click on a query box to go to that chat on page.
